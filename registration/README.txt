INSTALLATION

------------


You shall see the following files and directories:


auth.php              file with automatisation form
automatisation.php    file with automatisation form processing
registrationForm.php  file with registration form
registration.php      file with registration form processing
personal_room.php     file with user's personal account
connect.php           connect to DB
site.sql	      DB
README.txt            this file

To get started, run to file auth.php;don't forget to install DB.
